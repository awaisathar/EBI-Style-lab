{{> masthead-compact}}
De-compact the masthead {{#markdown}} This pattern works by tweaking the standard `.masthead-inner`. To enabele we add the responsive `.columns` classes and `.compact-for-data` is added to `header.masthead`. We can also add `.compact-disabled` to dynamically disable the responsive nature. Some guidance: - Only use this pattern on data orient pages. So don't use it on the landing page or about page. - We don't recommend using the `data-sticky-container` on the masthead, after all the focus is meant to be on the data. - Tou may want to add `.colapse` to `.masthead-inner.row` to remove margins on the child columns. {{/markdown}}
